# Environment

放 Vitis、Docker、WSL、服务器、license、路径和环境检查记录。

适合放：

- Vitis 2025.2 安装/可用性记录
- WSL/Docker 状态
- 远程服务器使用说明
- `vitis-run --mode hls` 检查记录
- 环境变量说明

不适合放：

- license server 密码
- SSH 私钥
- 个人账号 token

命名建议：

```text
YYYY-MM-DD-local-env-check.md
YYYY-MM-DD-vitis-server-setup.md
YYYY-MM-DD-docker-wsl-notes.md
```
