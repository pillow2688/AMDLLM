# Experiments

放模型、agent、prompt、budget policy、HLS 优化策略的实验记录。

每次实验必须记录：

- Date
- Task
- Agent version
- Model exact ID / provider
- Budget
- Tool call counts
- Functional result
- Synth result
- Cosim result
- Latency / score
- 失败原因或改进结论

不适合放：

- 没有上下文的零散截图
- 大量原始日志
- 私密 API 返回内容

命名建议：

```text
YYYY-MM-DD-dotproduct-model-comparison.md
YYYY-MM-DD-cosim-policy-ablation.md
YYYY-MM-DD-prompt-template-test.md
```
