# Official Materials

放官方规则、比赛页面、FAQ、邮件通知、submission 要求和关键日期记录。

适合放：

- FPT 2026 Track A 官方页面摘要
- 报名/提交/shortlist/demo 日期
- 官方 FAQ
- 官方邮件或通知摘要
- contest rule 变更记录

不适合放：

- 模型排行榜
- 个人理解笔记
- harness 代码阅读笔记

命名建议：

```text
YYYY-MM-DD-track-a-official-summary.md
YYYY-MM-DD-submission-requirements.md
```
