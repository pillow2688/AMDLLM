# Paper And Demo

放论文、答辩、demo 和最终提交材料的过程记录。

适合放：

- 2 页主文档大纲
- appendix 材料
- workflow 图
- token consumption 表
- demo script
- Q&A 准备

不适合放：

- 未整理的工具日志
- 模型 API key
- 与论文无关的基础教程

命名建议：

```text
YYYY-MM-DD-paper-outline.md
YYYY-MM-DD-token-consumption-table.md
YYYY-MM-DD-demo-script.md
```
