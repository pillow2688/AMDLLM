# Harness Notes

放 `fpt26-harness` 的阅读笔记、文件地图、运行方式、问题记录和本地修改说明。

适合放：

- `agent.py` 阅读笔记
- `harness.py` / `tools.py` / `scoring.py` 文件说明
- task package 格式说明
- sample task 运行记录
- harness 与比赛规则的对应关系

不适合放：

- 大量运行输出
- 复制完整源码
- 与 harness 无关的 HLS 教程

命名建议：

```text
YYYY-MM-DD-agent-py-reading.md
YYYY-MM-DD-toolserver-budget-notes.md
YYYY-MM-DD-sample-task-run-log-summary.md
```
