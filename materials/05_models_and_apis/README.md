# Models And APIs

放开源/开放权重模型、API provider、OpenRouter、license 和排行榜记录。

适合放：

- 模型候选列表
- OpenRouter model slug
- license 证据
- DataLearner / SWE-bench / Aider / BigCodeBench 排名快照
- provider 价格、上下文长度、稳定性记录

不适合放：

- API key
- 账号信息
- 没有来源日期的模型结论

命名建议：

```text
YYYY-MM-DD-open-model-shortlist.md
YYYY-MM-DD-openrouter-model-slugs.md
YYYY-MM-DD-model-license-evidence.md
```
